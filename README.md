# Library Web App

## Description
A simple interactive web application that simulates a library system where users can view, add, and manage books using HTML, CSS, and JavaScript.

## Live Demo 
https://jimmyhyusein.github.io/Library/

## Features
- Display list of books
- Add new books with details (title, author, etc.)
- Interactive UI with dynamic updates
- Responsive layout for a better user experience

## Technologies Used
- HTML
- CSS
- JavaScript
